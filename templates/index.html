Clique no botão para ver uma citação do Sasuke

Uso da API

GET /sasuke/quote

Obter uma citação aleatória do Sasuke

curl -X GET http://localhost:5000/sasuke/quote

GET /sasuke/quotes

Obter todas as citações do Sasuke

curl -X GET http://localhost:5000/sasuke/quotes

GET /sasuke/quote/:id

Obter uma citação específica pelo ID

curl -X GET http://localhost:5000/sasuke/quote/1

GET /sasuke/search?q=termo

Pesquisar citações contendo o termo

curl -X GET http://localhost:5000/sasuke/search?q=vingança

GET /sasuke/quotes/category/:categoria

Filtrar citações por categoria

curl -X GET http://localhost:5000/sasuke/quotes/category/Shippuden

POST /sasuke/quotes

Adicionar uma nova citação do Sasuke (requer autenticação)

curl -X POST http://localhost:5000/sasuke/quotes -H "Authorization: Bearer sasuke_api_token" -H "Content-Type: application/json" -d '{"quote": "Sua nova citação", "context": "Contexto", "source": "Episódio X", "category": "Categoria"}'

DELETE /sasuke/quote/:id

Remover uma citação do Sasuke (requer autenticação)

curl -X DELETE http://localhost:5000/sasuke/quote/1 -H "Authorization: Bearer sasuke_api_token"

PUT /sasuke/quote/:id

Atualizar uma citação existente (requer autenticação)

curl -X PUT http://localhost:5000/sasuke/quote/1 -H "Authorization: Bearer sasuke_api_token" -H "Content-Type: application/json" -d '{"quote": "Citação atualizada", "context": "Novo contexto"}'

POST /sasuke/favorites/:id

Adicionar uma citação aos favoritos (requer autenticação)

curl -X POST http://localhost:5000/sasuke/favorites/1 -H "Authorization: Bearer sasuke_api_token"

GET /sasuke/favorites

Listar todas as citações favoritas

curl -X GET http://localhost:5000/sasuke/favorites
